Reflection & Learnings

My Reflection & Learnings


In this course, I was able to learn managing enterprise servers under the culture of DevOps environment and the essentials of implementing DevOps environment in Linux servers like Ubuntu and CentOS, configuring and managing different services, monitoring availability, perfomance, and log of system, and using Ansible as the management configuration for automation. Learning GitHub is essential in this course as it stores your configuration file for future purposes. Through this course, I have learned to develop workflows using Infrastructure as code practice in developing an enterprise environment. I was able to know the processes and function of the real-world enterprise scenarios and how to impore the functionality if services. This course was fun and challenging to learn, aside from learning from our profressor, I was also able to learn how to debug problems in the system through learning form the problems and errors encountered on the activities we have done. Overall, this course helped me to apply my knowledge from the System Administration 1 basics and this knowledge will be used more in the future. I would like to thank our professor Dr. Jonathan V. Taylar for educating us about this course.
